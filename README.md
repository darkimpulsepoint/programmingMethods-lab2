# Лабораторная работа 2
### (ДЗ #2)
### Вариант 14

### Условия задач 

1. Написать программу вычисления дохода по вкладу. Пользователь может указать сумму и срок вклада. Процентная ставка зависит от суммы. Если сумма меньше 5000.00 р., то ставка составляет 10%. В противном случае ставка 13%.

2. Написать программу вычисления стоимости печати документов. Варианты печати - чёрно-белая, цветная. Если посетитель собирается напечатать больше 50 чёрно-белых страниц или 25 цветных, то ему предоставляется скидка 5%.

Предложения для задач 1,2: найти стандартную функцию библиотеки <cmath>, с помощью которой можно было бы выводить отдельно рубли и отдельно - копейки.

3. Написать программу для вычисления корней квадратного уравнения. Предусмотреть проверку отрицательного дискриминанта.

4. Написать программу, начисляющую кэшбек от покупки товара на сумму, указанную пользователем. Если пользователь купил на сумму больше 1000 р. - кэшбек составит 15%, если же на сумму больше 2000 р. - все 20%

5. Решить задачу из файла. Задачи в пределах подгруппы не должны повторяться! В файле с программой должен быть указан вариант
